solution

  • 首先，区间长度等于 111 是不可能的，具体可以看这篇题解。
  • 当区间长度为 222 时，由于相邻的两个数互质，所以左端点 xxx 满足 x(x+1)=mx(x+1)=mx(x+1)=m 此时化简方程得 x2+x−m=0x^{2}+x-m=0x2+x−m=0 用二次方程求根公式取较大的 −b+b2−4ac2a\frac{-b+\sqrt{b^{2}-4ac}}{2a}2a−b+b2−4ac​​ 得 x=−1+1+4m2x=\frac{-1+\sqrt{1+4m}}{2}x=2−1+1+4m​​ 此时判断是否为整数即可。
  • 当区间长度大于 222 时，我可以暴力枚举左端点和右端点预处理存在 map 中。
